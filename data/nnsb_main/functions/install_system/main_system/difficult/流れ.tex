前提条件
「通常」「お試し」「練習」の3つのモードが有る

説明
通常モード
・デフォルト設定
・アイテムロストあり

練習モード
・アイテムロストあり
・敵弱体化

お試しモード
・アイテムロストなし
・敵弱体化

流れ
看板クリック
チャット欄に現在のモード以外のモードが表示される。
チャット欄のモードをクリックすると
　「モード変更が申請されました。\n一度変更されると再度変更することはできません!!\n拒否する際は1分以内に拒否してください」

30秒経つと
　「30秒経ちました。」

10秒経つと
　「残り10秒です。」

5秒経つとカウントダウン

0秒になると
　「〇〇モードに切り替わりました。」
